# Network Scanner Python

Herramienta educativa desarrollada en Python para explorar conceptos básicos de redes y ciberseguridad.

## Objetivo

El objetivo de este proyecto es crear un escáner de red básico para comprender cómo funcionan las conexiones TCP/IP, los puertos y los servicios disponibles dentro de una red.

## Tecnologías utilizadas

- Python 3
- Sockets
- TCP/IP
- Git
- GitHub

## Estado del proyecto

🚧 En desarrollo

Actualmente cuenta con una estructura inicial. Próximamente incorporará detección de hosts y escaneo de puertos.
